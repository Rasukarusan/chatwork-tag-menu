# ChromeExtension-ChatworkTagMenu

## Overview

Chrome拡張。
Chatworkに[info]や[code]タグを挿入するショートカットメニューを追加する。

## Description

挿入できるのは以下のタグ
- [info][title][/title][info]
- [code][/code]
- 【やりたいこと】(これはオリジナル。これをキーにGASで収集するために追加)
